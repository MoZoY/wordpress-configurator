A WordPress plugin to automate initial configuration: general settings, permalinks, and plugin management.

== Description ==

Naro Configurator helps you quickly set up a new WordPress site:
- Change site title, tagline, URL, and admin email.
- Apply French presets (language, timezone, date/time format, permalinks).
- Install, activate, deactivate, or uninstall selected plugins, including support for ZIP-uploaded pro plugins.

== Installation ==

1. Upload the plugin ZIP via the WordPress admin or copy the `naro-config` folder to `wp-content/plugins/`.
2. Activate the plugin.
3. Go to "Naro Config" in the admin menu.

== Frequently Asked Questions ==

= Can I add my own plugins? =
Yes, add plugin ZIPs to `naro-config/plugins/` and they will appear in the admin interface.
